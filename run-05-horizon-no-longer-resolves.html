Transcript

the horizon no longer resolves if flickers at

the edge of perception as if the world

is failing to fully render itself powerfully and

in frozen collapse their shadows stretching in directions

that no longer correspond to light there's no

movement only the suggestion of it repeating the

system is not broken it's simplified sound has

been reduced to a low continuous presence not

a silence something beneath it a baseline frequency

that replaces wind replaces voice replaces thought it's

not fluctuate it's not respond it exists as

confirmation that something is still processing the environment

has been reclassified rain once defined by structure

and tension now exists as uniform output now

it's stayed nowhere because nowhere remains landsmarks persist

only as corrupted references struck meaning the left

intact for stability
